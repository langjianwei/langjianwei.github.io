Spring Boot快速入门之Hello Word（一）

发表于 2017-09-15     阅读次数： 次
  • Spring Boot是由Pivotal团队提供的全新框架，其设计目的是用来简化新Spring应用的初始搭建以及开发过程。该框架使用了特定的方式来进行配置，从而使开发人员不再需要定义样板化的配置。通过这种方式，Boot致力于在蓬勃发展的快速应用开发领域（rapid application development）成为领导者。

  • 大家在第1次接触或者学习Spring框架的时候，是否因为其繁杂的配置而退却了？反正我是这样，还记得那时候一脸懵逼的样子，在你第n次使用Spring框架的时候，是否觉得一堆反复黏贴的配置有一些厌烦？今天，教给大家一种全新的方式来使用Spring Boot，他让你更易上手，更简单快捷地构建Spring应用！

  • Spring Boot的主要优点：

    • 为所有Spring开发者更快的入门
    • 开箱即用，提供各种默认配置来简化项目配置
    • 内嵌式容器简化Web项目
    • 没有冗余代码生成和XML配置的要求

  • 今天两种方法教给大家构建Spring Boot应用 ：
    一 、访问Spring官网构建项目：
    1 .https://start.spring.io
    2 .如图所示：选择构建工具Maven Project、Spring Boot版本1.5.7以及一些工程基本信息，切记务必引入web模块

    3 .点击Generate Project下载项目压缩包
    4 .把下载的压缩包解压到eclipse的工作空间目录，右键导入已存在的maven项目，如图：

    5 .导入的项目结构如下：

      • src/main/java下的程序文件：我们的代码都在这个目录下面写
      • src/main/resources下的资源文件文件：存放我们的配置文件，静态，js，css样式文件等
      • src/test/下的测试文件：我们的单元测试代码
      • pom.xml文件：maven用于管理我们的项目jar包，上面第二部引入的web模块jar包，连接数据库的mysql等都在这里面引用

    6 .编写代码：

    • 在src/main/java下面新建一个controller包和一个类，如图：
    • 以前熟悉Spring MVC的同学都知道接下来改怎么做，Spring Boot简单的一行代码，两个注解完美解决：
      @RestController
public class HelloController {
    @RequestMapping("/hello")
    public String hello(){
         return "hello，郎建伟";
    }
}
    • @RestController注解用于返回一个json格式的字符串，相当于Spring MVC的@Controller和@ResponseBody两个注解加起来的作用
    • @RequestMapping注解用于映射我们的请求访问路径
